## リスクと対策

| リスク | 対策 |
|--------|------|
| クロスプラットフォーム問題 | 早期にCI/CDで検出 |
| 画像認識の精度低下 | 既存テストで回帰検出 |
| 破壊的変更 | フィーチャーブランチで開発 |
| ドキュメント遅延 | コード完成と並行で進行 |
